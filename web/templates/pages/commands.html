COMMAND REFERENCE

Slash Commands Specification

Detailed interface reference for all interaction endpoints exposed by gobot on Discord.

/ask ALL MEMBERS

Queries the configured AI provider and returns a synthesized response. If the AI determines moderation action is warranted, it initiates the confirmation flow.

>_
User
/ask question: "What is the latest score for the match?"
AI
gobot BOT
The match ended in a 1 - 1 draw, with the equalizer scored in the 88th minute.
/setkey SERVER OWNER ONLY

Sets or updates the server-wide AI provider and registers the API key. Triggers an automated validation query to ensure connection integrity and lists available models.

/config SERVER OWNER ONLY

Configures server settings. Currently supports toggling multi_message mode (on/off) to split long AI responses across multiple sequential messages.

/ping ALL MEMBERS

Measures and outputs the bot's current API latency and Discord gateway heartbeat delay.
